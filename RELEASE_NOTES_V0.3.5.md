# Car-Master v0.3.5 종합 안정화 릴리스 노트

## 사용자 관점에서 달라진 점

- 시공 요청을 연속으로 눌러 같은 거래가 중복 생성되는 상황을 방지했다.
- 거래 정보를 불러오지 못하면 빈 화면 대신 원인 안내와 다시 시도 버튼이 표시된다.
- 채팅을 보내는 동안 진행 상태가 보이며, 실패해도 작성한 글과 첨부파일이 유지된다.
- 비어 있거나 허용되지 않은 첨부파일을 업로드 전에 명확히 안내한다.
- 다른 탭에서 로그아웃하거나 세션이 바뀌면 현재 화면에도 반영된다.
- 같은 메시지가 중복 표시되거나 순서가 흔들릴 가능성을 줄였다.

## 유지된 부분

- 버텍스·후퍼옵틱·솔라가드·브이쿨·글라스틴트 등 기존 브랜드 기반 권장 시공 패키지 가이드와 가격 계산은 그대로다.
- 기존 화면, 메뉴, 라우팅, 데모 데이터, 회원가입과 관리자 승인 흐름은 유지된다.
- 결제, 정산, 지도 API, 새로운 서비스는 추가하지 않았다.

## 배포 전 주의

Supabase 마이그레이션은 이번 작업에서 실행하지 않았다. 실제 DB·Realtime·Storage 검증과 적용은 `SUPABASE_V0.3.6_CHECKLIST.md`에 따라 별도로 진행해야 한다.
