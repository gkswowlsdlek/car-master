# Car-Master v0.3.5 Baseline 릴리스 노트

## Added

- Supabase Auth 기반 실제 딜러·시공점·관리자 회원 구조
- 승인된 실제 시공점 조회 및 시공 요청 연결 구조
- 거래, 거래방, 채팅과 비공개 첨부파일의 Supabase 저장 기반
- 거래 참여자와 관리자 기준의 RLS 및 Realtime 기반
- 데모 계정 `1/1`, `2/2`, `3/3`을 이용한 역할별 QA 흐름

## Changed

- 실제 회원 데이터와 공개 데모 데이터를 분리했습니다.
- 실제 회원은 Supabase 저장소를 사용하고 데모 계정은 localStorage를 사용합니다.
- 가격 안내를 기존 브랜드 목록을 유지한 `권장 시공 패키지`로 정리했습니다.
- 프로젝트, 워크스페이스와 Footer의 버전 표기를 `v0.3.5`로 통일했습니다.

## Fixed

- 공개 페이지의 인증 초기화 지연이 무한 로딩으로 이어지지 않도록 보완했습니다.
- 시공 요청의 중복 생성 가능성을 줄였습니다.
- 채팅 메시지의 중복 표시와 순서 불안정 가능성을 줄였습니다.
- 비어 있거나 허용되지 않은 첨부파일과 10MB 초과 파일을 업로드 전에 차단합니다.

## 제외 범위

- 실제 결제·정산
- 지도 API와 정확한 거리 계산
- 관리자 공지
- 거래방 숨김·복원
- 읽음 표시, 답장과 메시지 검색

Supabase Migration은 자동으로 Production에 적용되지 않습니다. 실제 서비스 연결 전에는 환경 변수와 Migration 적용 상태를 별도로 확인해야 합니다.
