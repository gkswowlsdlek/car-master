## 역할·RLS

- [ ] 딜러 A는 자신의 거래만 조회·수정 가능
- [ ] 딜러 B는 딜러 A 거래 접근 거부
- [ ] 시공점은 배정된 거래만 조회·수정 가능
- [ ] 관리자는 전체 거래 조회 가능
- [ ] 승인 전·반려·정지 시공점은 검색 결과와 요청 대상에서 제외
- [ ] 채팅 `sender_id`와 `sender_role` 위조 거부
- [ ] 거래 참여자가 아닌 사용자의 거래방·메시지 접근 거부

## Realtime

- [ ] 서로 다른 브라우저/기기에서 새 메시지가 한 번만 표시
- [ ] 거래 상태 변경 실시간 반영
- [ ] 네트워크 끊김 후 재연결·정렬·중복 여부 확인
- [ ] 로그아웃 후 채널 구독 해제 확인

## Storage

- [ ] 허용 파일 업로드와 다운로드
- [ ] 0바이트·10MB 초과·미허용 MIME/확장자 거부
- [ ] 대문자 확장자·한글·긴 파일명 확인
- [ ] 참여자 외 서명 URL 생성·다운로드 거부
