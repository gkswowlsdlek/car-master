# Car-Master V0.3 Release Notes

V0.3은 새로운 기능을 늘리는 대신, 기존 시공 거래 흐름을 처음 접하는 사용자도 자연스럽게 이해하고 사용할 수 있도록 다듬은 Release Candidate입니다.

## V0.2.9에서 달라진 점

- 랜딩에서 카마스터가 카카오톡, 전화, 엑셀로 흩어진 업무를 어떻게 하나의 거래 흐름으로 정리하는지 더 명확하게 설명합니다.
- 가격 확인부터 시공 완료까지의 이용 과정을 4단계로 확인할 수 있습니다.
- 딜러 대시보드에서 오늘 입고 일정, 확인 대기, 진행 중, 완료 거래를 먼저 확인할 수 있습니다.
- 거래가 없을 때 첫 시공 요청으로 바로 이동할 수 있는 안내 화면을 제공합니다.
- 최근 거래, 최근 채팅, 최근 시공 요청을 한 화면에서 확인할 수 있습니다.
- 가격 가이드의 상품, 브랜드 필터, 차량 등급 선택 상태가 더 분명하게 표시됩니다.
- 시공 요청서의 필수 항목과 정보 구역을 구분하고, 선택한 시공점을 더 눈에 띄게 표시합니다.
- 시공점 카드의 거리, 평점, 최근 거래, 응답속도 정보를 읽기 쉽게 정리했습니다.
- 거래 진행 상태를 단계형 타임라인으로 개선해 현재 위치와 완료 단계를 한눈에 확인할 수 있습니다.
- 마이페이지에서 총 거래, 이번 달 거래, 완료 거래, 즐겨찾는 시공점 요약을 확인할 수 있습니다.
- 버튼, 카드, 입력창, 여백, 글자 크기와 반응형 레이아웃을 제품 전반에서 통일했습니다.
- 과하지 않은 Hover, 선택 강조, 전환 효과를 추가했습니다.

## 기존 기능 호환

V0.2.9의 로그인, 가격 조회, 시공점 검색, 시공 요청, 거래방 생성, 거래 채팅, 결제 상태, 프로필 저장 흐름은 그대로 유지됩니다. 신규 서버 API나 데이터 모델은 추가하지 않았으며, 화면의 활동 정보는 기존 거래와 시공점 데이터에서 계산합니다.
