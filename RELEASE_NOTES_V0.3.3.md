# Car-Master v0.3.3 — 시공점 워크스페이스 개선 패치

## Installer Workspace 2.0

- 시공점 홈에서 오늘 입고 예정, 현재 시공 중, 출고 대기 준비 상태와 오늘 완료 현황을 확인할 수 있습니다.
- 오늘, 내일, 이번 주 일정별로 차량과 시공 품목, 딜러, 현재 상태를 확인할 수 있습니다.
- 신규 시공 요청을 별도 영역에서 빠르게 확인하고 해당 거래방으로 이동할 수 있습니다.
- 기존 프로필을 확장해 휴무일과 긴급 시공 가능 여부를 관리할 수 있습니다.

## Transaction Chat UI Upgrade

- Desktop 거래방을 거래 목록, 대형 대화 화면, 자동 작업 브리핑의 3단 구조로 개선했습니다.
- 메시지 발신자, 시간, 날짜 구분선과 넓은 말풍선을 적용했습니다.
- Enter 전송, Shift+Enter 줄바꿈을 지원하는 고정형 입력 영역을 추가했습니다.
- 이미지와 일반 파일을 선택하고 전송 전에 확인하거나 삭제할 수 있습니다.
- 이미지 메시지를 클릭하면 크게 확인할 수 있습니다.

## 첨부 Prototype 정책

- 첨부 파일 원본은 localStorage에 저장하지 않습니다.
- 브라우저 Object URL을 사용하므로 첨부 미리보기는 현재 브라우저 세션에서만 유효합니다.
- 실제 서비스에서는 `LocalAttachmentProvider`를 Cloud Storage 업로드 구현으로 교체해야 합니다.

## 호환성

기존 transactionId, 거래방 ID, 메시지 저장소와 거래 상태 전이 규칙은 변경하지 않았습니다. 기존 브라우저 데이터는 별도 마이그레이션 없이 계속 사용할 수 있습니다.
