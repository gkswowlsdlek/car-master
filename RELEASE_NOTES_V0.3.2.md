# Car-Master V0.3.2 - API 연결 전 마지막 패치

V0.3.2는 새로운 사용자 기능을 추가하는 버전이 아니라, 현재 데모 기능을 유지하면서 실제 API 연결을 준비하는 안정화 버전입니다.

## 사용자 관점

- 딜러, 시공점, 관리자 데모 로그인과 각 역할의 진입 화면은 이전과 동일합니다.
- 가격 확인부터 시공 요청, 거래방 생성, 채팅, 상태 및 결제 확인까지 기존 흐름을 유지합니다.
- 브라우저에 저장된 기존 거래와 프로필 데이터는 별도 변환 없이 계속 사용할 수 있습니다.
- 로그인 처리 중 상태와 잘못된 계정 안내가 더 명확해졌습니다.

## 서비스 기반 개선

- 인증 처리를 화면에서 분리해 향후 OAuth 또는 서버 인증으로 교체할 수 있습니다.
- 시공점 역할은 기존 `shop` 데이터와 호환하면서 서비스 계층에서는 `installer`로 표준화합니다.
- 모든 브라우저 저장은 기존 repository 계층을 통해 접근합니다.
- 거래, 채팅, 일정, 가격과 상태는 기존 `transactionId` 중심 구조를 유지합니다.
- 외부 API 키는 코드에 포함하지 않고 환경변수 이름만 템플릿으로 제공합니다.

## 다음 단계

실제 API 연결 시 서버 세션과 권한 검증, 영구 데이터베이스, 지도 검색, 실시간 채팅 및 결제 제공자 구현을 현재 provider/repository 경계 뒤에 연결합니다.
