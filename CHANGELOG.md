# Changelog

## [0.2.1] - 2026-07-14

### Added
- 딜러 로그인 이후 전용 업무 대시보드 구조를 추가했습니다.
- 딜러 전용 좌측 고정 사이드바 메뉴를 추가했습니다.
- 대시보드 업무 요약 카드, 빠른 실행 메뉴, 패키지 홍보 배너를 추가했습니다.
- 최근 거래 목록과 오른쪽 정보 영역을 추가했습니다.
- 딜러용 결제 및 정산 샘플 화면을 추가했습니다.
- 딜러 마이페이지 샘플 화면을 추가했습니다.
- 거래 채팅 화면을 여러 거래를 전환할 수 있는 업무형 구조로 개선했습니다.
- 진행 중 거래 화면에 상태별 필터 샘플 데이터를 추가했습니다.

### Changed
- 딜러 화면을 Car-Master의 남색, 흰색, 파란색 중심의 B2B 업무용 UI로 개편했습니다.
- 기존 시공 요청서 메뉴명을 `시공 요청`으로 정리했습니다.
- 데스크톱에서는 고정 사이드바, 모바일에서는 하단 메뉴 중심으로 반응형 구조를 조정했습니다.
- 프로젝트 버전을 `0.2.1`로 업데이트했습니다.

### Preserved
- 랜딩페이지, 시공점 화면, 관리자 화면의 기존 흐름은 유지했습니다.
- 실제 Supabase, PG, 지도 API, 실시간 채팅 연결 없이 샘플 데이터 기반 프로토타입 구조를 유지했습니다.

### Notes
- 이번 버전은 `Car-Master v0.2.1 — Dealer Dashboard` 개발 중간 저장 버전입니다.
- Git 커밋 메시지: `feat: Car-Master v0.2.1 dealer dashboard redesign (WIP)`
