# Changelog

## v0.2.5 UX validation update - 2026-07-16

### Changed
- `금액표` 화면을 딜러 업무 시작점인 `권장 시공가` 화면으로 개편했습니다.
- 권장 시공가에서 `견적 문의`와 `실제 시공 요청`을 분리해 같은 요청 구조로 이어지도록 개선했습니다.
- 시공 요청 화면의 추가 작업 입력을 자유 입력형 `추가 작업 문의`로 변경했습니다.
- 시공점 선택 카드를 별점, 동 단위 지역, 거리, 거래건수, 가능 작업 중심으로 정리했습니다.
- 거래방 상단에 자동 생성되는 `작업 브리핑` 카드를 추가했습니다.
- 거래 진행상태를 `접수`, `입고`, `시공중`, `완료` 버튼으로 변경할 수 있는 샘플 기능을 추가했습니다.
- 거래 내역을 `현재 진행`, `완료`, `취소` 탭 중심으로 정리했습니다.
- 시공점 화면을 `새 문의`, `진행중`, `완료` 업무 탭 중심으로 개선했습니다.
- 관리자 화면을 거래 수, 회원 수, 시공점 수, 오늘 거래 중심의 전체 현황 화면으로 단순화했습니다.
- 공통 카드, 버튼, 입력창, 모바일 반응형 스타일을 더 일관된 SaaS UI로 정리했습니다.

## v0.2.5 - 2026-07-14

### Added
- 딜러용 브랜드별 금액표 화면을 추가했습니다.
- 버텍스 900, 솔라가드 프리미엄, 브이쿨 K, 후퍼옵틱 프라임, 레이노 F95 샘플 상품을 추가했습니다.
- 차량 등급별 예상 금액 표시 구조를 추가했습니다.
- 금액표에서 시공 요청으로 상품 정보를 전달하는 흐름을 추가했습니다.
- 차량모델 기반 샘플 차량 등급 분류를 추가했습니다.
- 선택 상품 요약과 예상 금액을 시공 요청 화면에 표시했습니다.
- 관리자 금액표 관리 샘플 UI를 추가했습니다.

### Changed
- 딜러 사이드바에 `금액표` 메뉴를 추가했습니다.
- 대시보드 패키지 배너를 금액표와 연결했습니다.
- 작업내용 빠른 선택을 농도와 추가 작업 중심으로 개선했습니다.
- 거래 관리와 거래 채팅 상단에 상품 및 예상 금액 정보를 표시했습니다.
- 시공점 요청 처리 화면에 선택 상품과 예상 금액 정보를 표시했습니다.
- package.json 버전을 `0.2.5`로 업데이트했습니다.

### Fixed
- 가격 문의를 채팅으로 다시 해야 하는 흐름을 줄이고 금액표에서 요청으로 이어지게 개선했습니다.

## v0.2.4 - 2026-07-14

### Added
- 딜러 4개 항목 기반 시공 요청 흐름을 추가했습니다.
- 요청정보 입력, 시공점 선택, 요청 확인 3단계 구조를 추가했습니다.
- 작업내용 자유 입력과 빠른 선택 버튼을 추가했습니다.
- 입고예정일 미정, 이번 주, 다음 주 빠른 선택을 추가했습니다.
- 시공지역 기준 등록 시공점 선택 패널을 시공 요청 화면에 추가했습니다.
- 시공점 계정에서 작업완료 처리 샘플 버튼을 추가했습니다.
- 추후 기능을 정리한 `BACKLOG.md`를 추가했습니다.

### Changed
- 딜러 사이드바 메뉴명을 `내 시공 요청`, `거래 관리`, `결제 및 정산`으로 정리했습니다.
- 딜러 대시보드 문구와 업무 요약을 `시공점 확인중`, `진행중`, `작업완료` 3상태 중심으로 정리했습니다.
- `승인 시공점` 표현을 `등록 시공점` 또는 `사업자 정보 등록` 표현으로 변경했습니다.
- 거래 상태 표시를 3단계 MVP 상태로 단순화했습니다.
- README를 v0.2.4 기준 실행 방법, 데모 계정, 핵심 흐름 중심으로 업데이트했습니다.
- package.json 버전을 `0.2.4`로 업데이트했습니다.

### Fixed
- 시공점 수락 후 딜러 거래 목록 상태가 `진행중`으로 반영되도록 개선했습니다.
- 수락 전 거래 채팅이 비활성화되는 정책을 더 명확하게 표시했습니다.

## [0.2.1] - 2026-07-14

### Added
- 딜러 로그인 이후 전용 업무 대시보드 구조를 추가했습니다.
- 딜러 전용 좌측 고정 사이드바 메뉴를 추가했습니다.
- 대시보드 업무 요약 카드, 빠른 실행 메뉴, 패키지 홍보 배너를 추가했습니다.
- 최근 거래 목록과 오른쪽 정보 영역을 추가했습니다.
- 딜러용 결제 및 정산 샘플 화면을 추가했습니다.
- 딜러 마이페이지 샘플 화면을 추가했습니다.
- 거래 채팅 화면을 여러 거래를 전환할 수 있는 업무형 구조로 개선했습니다.
- 진행 중 거래 화면에 상태별 필터 샘플 데이터를 추가했습니다.

### Changed
- 딜러 화면을 Car-Master의 남색, 흰색, 파란색 중심의 B2B 업무용 UI로 개편했습니다.
- 기존 시공 요청서 메뉴명을 `시공 요청`으로 정리했습니다.
- 데스크톱에서는 고정 사이드바, 모바일에서는 하단 메뉴 중심으로 반응형 구조를 조정했습니다.
- 프로젝트 버전을 `0.2.1`로 업데이트했습니다.

### Preserved
- 랜딩페이지, 시공점 화면, 관리자 화면의 기존 흐름은 유지했습니다.
- 실제 Supabase, PG, 지도 API, 실시간 채팅 연결 없이 샘플 데이터 기반 프로토타입 구조를 유지했습니다.
