# Car-Master V0.3.1 Beta Candidate Release Notes

V0.3.1은 Car-Master를 외부 사용자 테스트에 가까운 업무 도구로 발전시키기 위한 데스크톱 우선 베타 후보 버전입니다.

## 사용자가 체감하는 변화

### 공통 워크스페이스

- 역할별 업무 메뉴와 현재 화면 제목이 명확해졌습니다.
- 서비스 상태, 베타 환경, 계정 정보를 상단에서 확인할 수 있습니다.
- 딜러는 어느 화면에서든 새 시공 요청으로 빠르게 이동할 수 있습니다.

### 딜러

- 기존 V0.3 대시보드, 가격 가이드, 시공점 검색과 요청 흐름을 그대로 사용할 수 있습니다.
- 거래관리에서 전체, 확인 대기, 진행 중, 완료 건수를 먼저 확인할 수 있습니다.
- 거래 상태와 다음에 확인해야 할 행동이 더 분명하게 표시됩니다.

### 시공점

- 로그인 후 거래 목록 대신 시공점 전용 대시보드가 표시됩니다.
- 신규 요청, 오늘 입고, 진행 중, 완료 거래를 한눈에 확인할 수 있습니다.
- 우선 처리할 거래와 다음 행동을 바로 확인하고 거래방으로 이동할 수 있습니다.

### 관리자

- 전체 거래 수보다 신규 요청과 장기 미응답 등 확인이 필요한 상태를 우선 표시합니다.
- 거래 검색, 상태 필터, 사용자·결제·최근 채팅 확인이 쉬워졌습니다.

## 베타 운영 범위

V0.3.1은 기존 브라우저 저장소와 초대형 데모 계정을 유지하는 베타 후보입니다. 실제 외부 사용자가 동시에 데이터를 공유하는 공개 베타 전에는 서버 데이터베이스, 실제 인증, 개인정보 정책과 배포 운영 구성이 추가로 필요합니다.

## 유지된 기능

- 역할별 로그인
- 차량 등급별 가격 가이드
- 행정구역 기반 시공점 검색
- 시공 요청 작성과 최종 확인
- 거래방과 작업 브리핑
- 거래별 채팅
- 역할별 거래 상태 변경
- 가격 확인과 결제 상태 기록
- 프로필과 활동 요약
