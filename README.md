# Car-Master 클릭형 MVP 프로토타입

리스·렌트 딜러와 고객 지역의 자동차 시공점을 연결하는 B2B 플랫폼의 1차 프론트엔드 프로토타입입니다.

## 현재 구현 범위

- 랜딩, 역할 선택 회원가입, 로그인
- 딜러: 차량 등록, 부산 시공점 검색, 패키지 선택, 요청, 일정 동의, 테스트 결제, 진행 상태
- 시공점: 정보 등록, 승인 상태, 패키지 참여, 요청 수락·거절, 일정 제안, 시공 상태, 정산 예정액
- 관리자: 시공점·패키지 참여 승인, 패키지, 이용료, 거래·결제·정산 목록
- 샘플 데이터: 딜러 5명, 시공점 10곳, 패키지 3개, 상태별 거래 10건

실제 Supabase, 지도 API, PG, 실시간 채팅, 파일 업로드, 송금은 연결하지 않습니다.

## 실행

Node.js 22.13 이상이 필요합니다.

```bash
npm install
npm run dev
```

화면에 표시되는 로컬 주소를 브라우저에서 엽니다. 로그인 화면에서는 입력값과 관계없이 딜러·시공점·관리자 역할을 선택해 시연할 수 있습니다.

## 핵심 시연 순서

1. 랜딩에서 `딜러로 시연하기`
2. 딜러 로그인
3. `시공점 찾기`에서 부산 오토랩 선택
4. 버텍스 500 패키지 선택 후 시공 요청
5. 좌측 하단 `시연 역할 전환`에서 시공점 선택
6. 신규 요청 수락 및 일정 제안
7. 딜러로 전환해 일정 동의 및 테스트 결제
8. 시공점으로 전환해 입고·시공·완료 상태 변경
9. 정산 화면 또는 관리자 이용료 설정에서 수수료와 정산액 확인

## 정책

- 패키지 가격은 부가세 포함 원화 정수
- 플랫폼 이용료 테스트 기본값 3%, PG 수수료와 별도 관리
- 두 수수료는 시공점 정산 예정액에서 차감
- 수수료는 원 단위 반올림
- 요청 응답 기한 기본 48시간
- 고객 이름·전화번호 미수집, 차량번호 선택
