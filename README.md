# Car-Master

Car-Master는 리스·렌트 에이전시 딜러와 자동차 시공점을 연결하고, 가격 확인부터 시공 완료까지의 업무를 거래 단위로 관리하는 B2B 플랫폼입니다.

현재 버전은 `Car-Master V0.3.1 Beta Candidate`입니다.

## 핵심 사용자 흐름

### 딜러

1. 시공 가격과 차량 등급별 가이드 금액 확인
2. 인도 지역 주변 시공점 검색 및 비교
3. 차량, 작업, 일정과 시공점을 선택해 요청 전송
4. 자동 생성된 거래방에서 일정, 채팅, 가격과 상태 확인

### 시공점

1. 대시보드에서 신규 요청과 오늘 입고 일정 확인
2. 우선 처리 거래와 다음 행동 확인
3. 거래방에서 작업 정보와 요청사항 확인
4. 입고예정, 입고, 시공중, 완료 단계 처리

### 관리자

1. 전체 거래와 확인 필요 상태 모니터링
2. 거래, 사용자, 시공점과 상태 검색
3. 결제 상태와 최근 채팅 기록 확인

## 실행 방법

Node.js 22.13 이상이 필요합니다.

```bash
pnpm install
pnpm dev
```

브라우저에서 `http://localhost:3000`을 엽니다.

## 베타 계정

| 역할 | 아이디 | 비밀번호 |
| --- | --- | --- |
| 딜러 | `1` | `1` |
| 시공점 | `2` | `2` |
| 관리자 | `3` | `3` |

## 검증 명령

```bash
npm test
npm run lint
npm run build
```

## V0.3.1 운영 범위

현재 거래와 프로필은 브라우저 localStorage에 저장됩니다. 실제 공개 베타 전에는 서버 데이터베이스, 실제 회원 인증, 계정별 접근 권한, 개인정보 처리방침과 배포 모니터링이 필요합니다.

자세한 변경 사항은 `RELEASE_NOTES_V0.3.1.md`를 참고하세요.
