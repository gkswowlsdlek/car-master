# Car-Master

Car-Master는 리스·렌트 에이전시 딜러가 차량모델, 시공지역, 작업내용, 입고예정일 4가지만 입력해 전국 등록 시공점에 시공 요청을 보내고 거래를 관리하는 B2B 업무 플랫폼 프로토타입입니다.

현재 버전은 `Car-Master v0.2.5 - Dealer Price Guide & Package Selection`입니다.

## 현재 구현

- 딜러 중심 랜딩페이지
- 이메일/비밀번호 방식 데모 로그인
- 딜러 대시보드
- 딜러 권장 시공가
- 브랜드별 신차패키지 샘플 상품 5종
- 국산 승용 기준 권장 시공가 구조
- 권장 시공가 상품 선택 후 견적 문의 또는 실제 시공 요청으로 연결
- 4개 항목 기반 내 시공 요청 흐름
- 추가 작업 문의 자유 입력
- 요청정보 입력 → 시공점 선택 → 요청 확인 3단계 구조
- 샘플 등록 시공점 약 100곳
- 거래 상태 3단계: 시공점 확인중, 진행중, 작업완료
- 거래 관리 목록과 현재 진행, 완료, 취소 탭
- 거래방 자동 작업 브리핑
- 접수, 입고, 시공중, 완료 진행상태 버튼
- 수락 전 비활성화되는 거래 채팅 UI
- 시공점 새 문의, 진행중, 완료 업무 화면
- 관리자 전체 현황 화면
- 관리자 금액표 관리 샘플 UI

## 실행 방법

Node.js 22.13 이상이 필요합니다.

```bash
pnpm install
pnpm dev
```

브라우저에서 `http://localhost:3000`을 엽니다.

## 데모 계정

| 역할 | 아이디 | 비밀번호 |

| 딜러 | `1` | `1` |
| 시공점 | `2` | `2` |

## 딜러 핵심 흐름

1. 로그인 후 대시보드에서 오늘 업무 상태 확인
2. `권장 시공가`에서 브랜드와 상품 검색
3. 국산 승용 기준 권장 시공가 확인
4. `견적 문의` 또는 `실제 시공 요청` 클릭
5. `내 시공 요청`에서 선택 상품 요약 확인
6. 차량모델, 시공지역, 작업내용, 입고예정일과 추가 작업 문의 입력
7. 시공지역 주변 등록 시공점 목록 확인
8. 시공점 선택
9. 요청 내용 최종 확인
10. 요청 전송
11. 거래방 생성 및 자동 작업 브리핑 확인
12. 시공점 수락 후 거래 채팅 활성화
13. 접수, 입고, 시공중, 완료 상태 확인

## 시공점 핵심 흐름

1. 새 문의, 진행중, 완료 업무 탭 확인
2. 자동 작업 브리핑에서 차량, 작업, 추가 작업, 입고예정일 확인
3. 요청 수락 또는 거절
4. 수락 후 거래 채팅 활성화
5. 작업완료 처리
